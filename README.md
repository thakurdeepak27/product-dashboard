# Product Dashboard

A **Product Management application** built with **React** and **Vite**, featuring full CRUD operations (Create, Read, Update, Delete) using a REST API.  
The app is fully **responsive**, includes **form validation**, and uses a clean **component-based architecture** with a separate `services` folder for API calls.

---

## **Features**

- Display a grid of products fetched from a REST API
- Add new products via a dynamic form
- Edit existing products
- Delete products with a confirmation modal
- Form validation with real-time error messages
- Smooth UX with loading and saving indicators
- Clean component structure (`ProductForm`, `ProductCard`, `ConfirmModal`)
- Environment-variable-based API URL for development and production

---

## **Tech Stack**

- **React** (Functional Components, Hooks)
- **Vite** (Fast bundler and dev server)
- **Axios** for HTTP requests
- **ESLint** for code quality
- **CSS** for styling

---

## **Installation / Setup**

Follow these steps to get the project running locally:

### **1. Clone the repository**

```bash
git clone https://github.com/YourUsername/product-dashboard.git
cd product-dashboard

Install dependencies
npm install
This will install:
React, React-DOM
Axios
Vite

. Set up environment variables
Create a .env file at the project root:
VITE_BASE_URL=https://fakestoreapi.com/products

Run the development server
npm run dev
```
